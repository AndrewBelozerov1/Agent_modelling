# Мультиагентное моделирование финансовых рынков

## О проекте

Данный проект представляет собой агентно-ориентированную модель (Agent-Based Model) финансового рынка, в которой взаимодействуют три типа трейдеров с различными торговыми стратегиями:

- **Шумовые трейдеры (Noise Traders)** — принимают случайные решения о покупке/продаже
- **Технические трейдеры (Chartists)** — используют скользящую среднюю для принятия решений
- **Фундаменталисты (Fundamentalists)** — оценивают справедливую стоимость актива на основе дивидендов и избыточной доходности

Модель позволяет исследовать механизмы ценообразования, влияние различных типов участников на волатильность рынка, а также тестировать регуляторные механизмы (остановку торгов и эволюцию стратегий).

## Ключевые выводы

Фундаменталисты являются наиболее устойчивыми — в большинстве сценариев они сохраняют или приумножают капитал;

Технические трейдеры чувствительны к волатильности — в базовом сценарии теряют заметную часть капитала, но при доминировании на рынке могут получать сверхприбыль;

Шумовые трейдеры создают ликвидность, но их избыток приводит к обвалу рынка;

Circuit Breaker (остановка торгов при изменении цены >20%) снижает хаотичность рынка;

Эволюция трейдеров (переход от простых стратегий к сложным) снижает волатильность рынка во второй половине симуляции.

## Используемые метрики

### Показатель Херста (H)

H = 0.5 — случайное блуждание (отсутствие памяти)

0.5 < H < 1 — персистентность (тренд сохраняется)

0 < H < 0.5 — антиперсистентность (возврат к среднему)

### Показатель Ляпунова (λ_max)

λ_max > 0 — детерминированный хаос (чувствительность к начальным условиям)

1/λ_max — горизонт планирования (в днях)

### Энтропия Колмогорова-Синая (h_KS)
